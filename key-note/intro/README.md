# 介绍

以Quarks作为开发框架，开发基于云原生的应用程序，包括MicroProfile规范API的使用和在Kubernetes云平台中部署Quarkus应用程序。

## Cloud-Native

* 云原生是基于分布部署和统一运管的分布式云
* 以容器、微服务、DevOps等技术为基础建立的一套云技术产品体系。
